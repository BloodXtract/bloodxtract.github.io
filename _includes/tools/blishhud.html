BlishHUD is a modular overlay for Guild Wars 2. It functions using modules, which can be installed through "Module Repo" in BlishHUD windows.

BlishHUD is an external application, and needs to be started through its .exe separately from Guild Wars 2 - but it is strongly recommended!

Some of the modules we recommend all members to install:

  • Discord Rich Presence - will display your current character and location in Discord.
  • Events and Metas Observer - will display a notification when world events are about to start. You can selectively select which events you want to be notified of using a configuration tab on the very left of BlishHUD window!
  • Clock - displays a customizable clock on screen, that can be dragged around, resized, and also show Local, Server and Tyria (in-game) time all at once.
  • Regions of Tyria - a small module that shows name of the region of the map as you enter it.
  • Universal Search - adds a button on top of your screen that allows you to search for any location (such as Waypoint, POI or Vista) in the game and copy its link so you can paste it into the chat.
  • Distance Rings - mainly for ranged combat - allows you to create up to 5 rings at your character feet at distance - allowing you to easily see if your targets are within skills range.

Once you install a module, remember to enable it through "Manage Modules" tab - some also have additional settings!
